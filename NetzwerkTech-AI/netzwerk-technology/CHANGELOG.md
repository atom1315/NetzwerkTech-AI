# Changelog

Alle wichtigen Änderungen an diesem Projekt werden hier dokumentiert.

## [2.0.0] — 2026-03-26

### Hinzugefügt
- 🧠 **Graph Neural Network** Topologie-Analyse mit GraphSAGE
- ⚡ **Reinforcement Learning** Auto-Optimierung mit PPO
- 🔮 **Prädiktive Wartung** mit GRU + Bayesian Inference
- 📊 **Echtzeit-Dashboard** mit React + D3.js + WebSocket
- 🔌 **GraphQL API** für flexible Datenabfragen
- 🐳 **Docker Compose** Setup mit TimescaleDB, Redis, Kafka
- 📈 **Performance-Benchmarks** gegen Snort, Suricata, Zeek
- 🌐 **Topologie-Visualisierung** mit interaktivem Graph-Editor

### Verbessert
- Anomalie-Erkennung F1-Score von 89% auf 96.2%
- Inference-Latenz von 45ms auf 12ms reduziert
- Durchsatz von 15 Gbps auf 40 Gbps erhöht
- Mixed Precision (FP16) Support für schnelleres Training

### Geändert
- Migration von TensorFlow auf PyTorch als primäres Framework
- Neue Microservice-Architektur ersetzt monolithisches Design
- TimescaleDB ersetzt InfluxDB für Zeitreihendaten

## [1.0.0] — 2025-09-15

### Hinzugefügt
- 🔍 Anomalie-Erkennung mit Autoencoder + LSTM
- 🛡️ Intrusion Detection System (Transformer-basiert)
- 📊 Traffic-Analyse mit CNN + Attention
- 🔌 REST API mit FastAPI
- 📖 Dokumentation und Beispiele
